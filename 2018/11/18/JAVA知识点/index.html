1.下列说法错误的有（ ）

解析：

java不允许单独的方法，过程或函数存在,需要隶属于某一类中。——AB错

java语言中的方法属于对象的成员,而不是类的成员。不过，其中静态方法属于类的成员。——C错

java需要实例化对象才可以访问非静态方法或者直接类名访问静态方法，而c/c++不需要对象就可以访问 ——D错
